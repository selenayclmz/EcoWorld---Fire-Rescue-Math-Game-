## About the Project
EcoWorld is an educational math game designed for primary school students.
The player controls a plane and helps save the city from fires by solving math questions.

## Purpose
The purpose of this game is to teach basic addition and subtraction.
It also teaches cause and effect.
Correct answers extinguish fires, while wrong answers make fires grow.

## How to Play
- Move the plane using the mouse.
- Catch water orbs with numbers.
- Click to drop water on fires.
- If the number matches the math question, the fire is extinguished.
- If too many buildings collapse, the game is over.

## Technologies Used
- Java
- Java Swing
- Timer (Game Loop)
- Mouse Events

## Game Mechanics
- Fires appear randomly on buildings and trees.
- Each fire has a math question.
- Fires grow step by step if not extinguished.
- The player wins by reaching the target score.
